Flashcards Sobre Biologia: Para fins de ensino.

O projeto é uma página web criada com HTML, CSS e JavaScript, destinada a exibir um conjunto de **flashcards de biologia**. O objetivo é criar uma plataforma interativa onde seja possível estudar conceitos importantes de biologia de forma prática e dinâmica, permitindo revisar tópicos como anatomia, ecologia e genética.

A estrutura do site é baseada em HTML para definir a semântica e o layout do conteúdo, enquanto o CSS estiliza a página com cores e tipografia apropriadas ao tema de biologia. O JavaScript é responsável pela interatividade dos flashcards, como virar para revelar respostas, navegar entre os cartões e armazenar progresso do estudo.

O HTML organiza o conteúdo em um cabeçalho com o título do site, seguido por uma divisão principal (.container) que apresenta uma introdução ao propósito do site e a coleção de flashcards em uma seção central (.flashcards). Cada flashcard é exibido em seu próprio contêiner (.card), mostrando a frente com uma pergunta ou conceito e, ao ser clicado, a parte de trás com a resposta ou explicação. O design responsivo é garantido pelo uso da meta tag viewport e por layouts flexíveis que se adaptam a diferentes dispositivos.

O CSS personaliza o visual do site, utilizando variáveis de cores inspiradas na natureza e uma tipografia clara importada do Google Fonts, garantindo uma atmosfera educacional e relaxante. Já o JavaScript é essencial para adicionar funcionalidades dinâmicas, como animações ao virar os cartões, filtragem por temas e opções para acompanhar o desempenho do usuário. Essa combinação de tecnologias proporciona uma experiência interativa, educativa e visualmente atraente, incentivando o aprendizado contínuo de biologia.

N-Space: A Loja de Jogos Eletrônicos Oficial da Nintendo.

O projeto é uma página web criada com HTML e CSS, destinada a exibir uma loja oficial fictícia de jogos de videogame da Nintendo. O objetivo é oferecer um ambiente virtual onde os usuários possam explorar os jogos disponíveis, visualizar detalhes como título, desenvolvedor, ano de lançamento e ter a opção de compra direta.

A estrutura do site é baseada em HTML, que organiza o conteúdo e define a semântica da página, enquanto o CSS é usado para criar um design visual atrativo, incluindo cores, tipografia e disposição dos elementos. O objetivo é proporcionar uma navegação fluida e agradável.

O HTML estrutura o conteúdo com um cabeçalho que apresenta o logotipo da loja e um menu de navegação, seguido por uma divisão principal (.container) que destaca os jogos em destaque e oferece uma galeria organizada (.jogos) para exibir o catálogo completo. Cada jogo é apresentado em um contêiner individual (.jogo), com imagem da capa, informações relevantes e um botão que direciona para a página de compra. A responsividade é assegurada pelo uso de uma meta tag viewport e um layout flexível que se adapta a diferentes dispositivos.

O CSS define a identidade visual da loja, usando variáveis para criar um esquema de cores que remete ao universo Nintendo, tipografia moderna importada do Google Fonts e botões estilizados para interatividade. O design visual foi pensado para garantir acessibilidade e tornar a experiência do usuário intuitiva. As escolhas de cores e fontes refletem o espírito vibrante e divertido da marca, incentivando a navegação e o interesse pelos jogos disponíveis.

Mega Duolingo: Divulgação de um Plano de Assinatura.

O projeto é uma página web criada com HTML e CSS, destinada a divulgar um plano especial do Duolingo. O objetivo é criar um ambiente virtual atrativo, onde os visitantes possam conhecer os benefícios do plano, como recursos exclusivos, aprendizado acelerado e uma experiência livre de anúncios, incentivando a assinatura.

A estrutura do site é baseada em HTML, que organiza o conteúdo de forma clara e intuitiva. O CSS estiliza a página visualmente, destacando elementos-chave, como o título, os benefícios do plano e os botões de chamada para ação (CTAs), utilizando cores vibrantes e tipografia moderna.

O HTML estrutura o site com um cabeçalho que apresenta o logotipo e o slogan do Duolingo, seguido por uma seção principal (.container) que explica os diferenciais do plano. Nesta seção, os recursos são apresentados em blocos organizados (.beneficio), incluindo detalhes como lições personalizadas, práticas offline e suporte prioritário. Um botão destacado conduz o visitante diretamente para a página de assinatura.

O CSS aprimora a estética do site, utilizando variáveis para um esquema de cores dinâmico que remete à identidade visual do Duolingo, fontes amigáveis importadas do Google Fonts e animações suaves para os botões. A meta tag viewport e o layout flexível garantem que o site seja responsivo e ofereça uma experiência otimizada em dispositivos móveis. O design foi pensado para capturar a atenção do usuário e transmitir a proposta do plano de forma clara, convidando à assinatura.
